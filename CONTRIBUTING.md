# Contributing

This knowledge base is maintained by the Center for Cooperative Media team. Contributions that improve accuracy or add missing information are welcome.

## Adding a staff profile

Create a new file at `ccm-profiles/staff/firstname-lastname.md` using this template:

```markdown
# Full Name

**Title:** ...
**Email:** ...@montclair.edu
**Twitter:** [@handle](https://twitter.com/handle)
**Booking:** [Schedule a meeting](calendly-url)
**MSU profile:** [montclair.edu/profilepages](url)

---

## Background
## Education
## Areas of focus
## Role at CCM
```

Source information from the [CCM website](https://centerforcooperativemedia.org/), MSU profile pages, and LinkedIn.

## Adding a project profile

Create a new file at `ccm-profiles/projects/kebab-case-name.md` using this template:

```markdown
# Project Name

**Type:** ...
**Scope:** ...
**Website:** [display-url](full-url)
**Members:** ...
**Cost to join:** ...
**Contact:** ...

---

## Overview
## Key features
## History
## Funders
## Staff leads
```

## Keeping cross-references in sync

Staff profiles reference the projects they lead. Project profiles list their staff leads. When adding or editing either, check the other side and update if needed.

## Style notes

- Use sentence case for headings (not Title Case)
- Staff filenames: `firstname-lastname.md`
- Project filenames: `kebab-case-name.md`
- All emails use `@montclair.edu`
